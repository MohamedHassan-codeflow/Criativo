# Criativo - Creative Agency Website

Criativo is a modern and responsive Creative Agency landing page designed and developed using HTML5 and CSS3.  
The project focuses on delivering a professional dark-themed UI with smooth animations, responsive layouts, and visually appealing sections for agencies, portfolios, and creative businesses.

## 🌐 Live Demo

👉 https://criativo-weld.vercel.app/

---

## 🚀 Features

- Fully Responsive Design
- Modern Dark UI/UX
- Smooth Hover Animations
- Interactive Service Cards
- Creative Agency Hero Section
- About Us Section
- Portfolio Gallery
- Latest Projects Showcase
- Team Members Section
- Testimonials Section
- Blog & News Cards
- Contact Form
- Social Media Footer
- Optimized for Multiple Screen Sizes
